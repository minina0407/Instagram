# PortfoGram
## 프로젝트 소개
"인스타그램+ 포트폴리오" = 포트폴그램 <br>
개발자를 위한 소통 SNS 플랫폼 입니다. <br> 
자신의 포트폴리오를 자랑하고, 다른 사람들의 포트폴리오를 구경하며, 커피챗 등을 통해 개발자들의 인맥도 넓힐 수 있는 플랫폼입니다. <br>
현재는 API 개발 진행중으로, API만 개발 중이고, 프론트엔드는 개발하지 않은 상태입니다.
